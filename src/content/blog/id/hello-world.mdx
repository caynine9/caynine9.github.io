Welcome to my new blog powered by **Astro**, **Svelte**, and **Tailwind CSS**. 

## Why this stack?

Astro provides an amazing developer experience for content-focused websites. It ships **zero JavaScript** by default, meaning our blog will be blazing fast. 

We can also mix and match components from any framework. Here, we are using Svelte to build interactive elements when needed, styled effortlessly with Tailwind CSS.

> The design language is inspired by Apple's clean, minimalist aesthetics, featuring the SF Pro font and smooth glassmorphism effects.

### Interactive Components

Because we are using MDX, we can even embed interactive Svelte components directly inside our markdown files!
